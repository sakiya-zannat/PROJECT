# Qt E-Commerce Desktop Application

A comprehensive desktop-based E-Commerce management application built using **C++** and the **Qt Framework**. This project demonstrates the implementation of Object-Oriented Programming (OOP) principles, clean UI/UX design, and database integration for managing products, users, and shopping carts.

---

## 🚀 Features

* **User Authentication:** Secure login and registration system for customers and administrators.
* **Product Catalog:** Interactive dashboard to browse products with filtering and search functionalities.
* **Shopping Cart & Billing:** Dynamic cart management (add, remove, update quantities) with automated total price calculation.
* **Admin Panel:** Specialized view for administrators to add, update, or remove products from the inventory.
* **Database Integration:** Persistent data storage using SQLite to manage user profiles and product details.

---

## 🛠️ Tech Stack

* **Language:** C++
* **Framework:** Qt Creator (UI/UX Design & Logic)
* **Database:** SQLite / Qt SQL Module
